# 折姜大学校规

## 1. 快速上手
- 通过入学考试进入校园，系统会分配专业与首学期课程。
- 在课程卡片选择“摆 / 摸 / 卷”策略，影响擅长度增长与精力消耗。
- 每个 Tick 约 3 秒自动结算：课程进度、精力、心态、压力、随机事件。
- 期末考试根据擅长度、心态、压力、运气给出成绩并更新 GPA。
- 精力或心态归零会 Game Over；完成 8 学期则毕业并生成文言文总结。

## 2. 基础设定
- Tick 时长：约 3 秒；暂停时不结算，不触发事件/钉钉。
- 学期计时：底部倒计时为当前学期剩余时间；结束后可进入下一学期。
- 课程权重：学分越高，对总精力消耗占比越大。
- 专业与课程：每学期自动分配，与专业课表、难度、学分相关。

## 3. 核心属性
- **精力**：随课程策略结算消耗；低消耗或摆烂可小幅回血；归零即 Game Over。
- **心态**：影响成长与考试；低于 50 减益，80+ 有加成；归零即 Game Over。
- **压力**：40-70 为最佳区间；区间内成长/考试有加成，过低/过高减益。
- **IQ / EQ / 运气**：IQ 影响学习效率；EQ 用于事件与社交；运气影响考试波动与事件。
- **声望**：部分事件与成就使用。

## 4. 课程策略（三态）
- **摆 (0)**：几乎不消耗精力，总消耗极低时会回少量精力并减压，但成长很慢。
- **摸 (1，默认)**：均衡成长，正常消耗；受心态/压力影响成长速度。
- **卷 (2)**：成长最快，精力消耗高，压力可能上升；同样受心态/压力影响。
- 成长算法：基础成长 × 策略系数 × (1 + (IQ − 100)%) × 心态/压力修正。

## 5. 休闲与冷却
- **健身**：需要较高精力（默认 ≥ 50）；消耗后大幅回精力，心态 +，压力 −。
- **开黑**：小幅耗精力，心态大幅回升。
- **散步**：降低压力。
- **刷 CC98**：随机正/负效果，可能回血心态或增加压力。
- 每个动作有独立冷却，冷却中会提示剩余秒数。

## 6. 期末考试与 GPA
- 单科分数 ≈ 擅长度 × 0.9 + 心态/压力修正 + 运气波动 + 基础分。
- 挂科阈值读取平衡配置，低于阈值计入挂科数并扣心态。
- 绩点 = max(分数/10 − 5, 0)，$\text{GPA} = \sum(\text{学分} \times \text{绩点}) / \text{总学分}$。
- 全科不挂会奖励心态；考试后生成成绩单并更新最高 GPA。

## 7. 随机事件与钉钉
- 随机事件：按配置的间隔与概率触发，基于当前状态生成剧情；选项会改动属性（有安全上限）。
- 钉钉消息：按间隔 + 概率推送，内容受心态/压力/GPA 影响（安慰、催促或祝贺）。
- 已出现的随机事件标题会记录，避免短期重复。

## 8. 学期推进与毕业
- 完成一学期后可进入下一学期，自动分配新课表并重置擅长度。
- 大一秋冬起计，共 8 学期；超过 8 学期则判为毕业并展示文言文总结与成就。
- 学期切换会尝试自动存档；假期可能插入假期事件。

## 9. 成就与结局
- 成就示例：GPA ≥ 4.0、心态跌破 10、EQ ≥ 95、学习次数 50+ 等，会即时解锁提示。
- 毕业时展示已解锁成就；失败（精力/心态归零）则进入 Game Over，可选择重开。

## 10. 友好提示
- 保持心态 > 50、压力在 40-70 区间，成长和考试都更稳。
- 精力告急时切换“摆”，配合散步/开黑回血，避免猝死。
- 冷却类休闲优先健身与开黑；CC98 有风险，低心态时慎点。
- 暂停后再操作更安全，防止误触浪费精力。
